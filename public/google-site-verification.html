Verificación de Google Search Console

Estado: Pendiente de configuración

Este archivo debe ser reemplazado con el archivo de verificación HTML proporcionado por Google Search Console.

Pasos para configurar:

  1. Accede a Google Search Console
  2. Agrega la propiedad: https://terapiadeparejamedellin.com
  3. Selecciona "Archivo HTML" como método de verificación
  4. Descarga el archivo y reemplaza este archivo completamente
  5. Verifica la propiedad en Google Search Console

Para más detalles, consulta: Guía completa de configuración
